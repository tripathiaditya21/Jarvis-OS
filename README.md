🤖 JARVIS OS

A futuristic AI Operating System inspired by Iron Man’s JARVIS, built with Python, FastAPI, Ollama, and Large Language Models.

🚀 Overview

JARVIS OS is a personal AI assistant designed to run locally on macOS using Ollama and Llama 3.2. It aims to become a complete AI operating system capable of conversations, memory, voice interaction, automation, coding assistance, and computer control.

✨ Current Features

* 🧠 Local AI using Ollama
* 🤖 JARVIS personality
* ⚡ FastAPI backend
* 💬 REST API chat endpoint
* 🔒 Offline inference (no API cost)

🛠️ Tech Stack

* Python
* FastAPI
* Ollama
* Llama 3.2
* SQLAlchemy (Upcoming)
* React + Tauri (Planned)
